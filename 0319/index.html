# Ubuntu명령어 - 현재 경로 출력: pwd - 파일과 디렉토리 출력: ls - 모든 파일과 디렉토리 출력: ls -a - 디렉토리 생성: mkdir (경로)(디렉토리명) [예: mkdir 1/2/3 == 1 아래의 2 아래에 3 디렉토리 생성] - 디렉토리 삭제: rmdir (디렉토리명) [디렉토리가 비어있어야 함] [파일 강제 삭제: rm -f, 파일 모두 강제 삭제: rm -rf] [실수로 루트를 영영보내지 않기 위해 웬만하면 경로에 ./를 꼭 달아주자] - 디렉토리 이동: cd (디렉토리명) - (없는)파일 생성, (있는)파일 타임스탬프 갱신: touch (경로)(디렉토리명) - 파일 복사: cp (복사할 파일) (저장할 디렉토리 경로) - 파일 이름 변경: mv (기존 이름) (변경 후 이름) [당연히 경로 설정 가능!] - cat 사용 중 나가기: ^D [Ctrl + D] - 출력 재지정: (명령어) > (파일) [명령어의 표준출력을 파일에 저장] [>는 파일의 내용에 새로 덮어 쓰고, 기존의 내용은 초기화된다.(즉, 재지정된다.)] - 출력 추가: (명령어) >> (파일) [명령어의 표준출력을 파일에 저장] [>>는 대상 파일이 이미 존재하는 경우, 그 파일의 내용에 추가한다.] - 후면처리: (명령어)$ [명령어를 후면에서 처리하여 그 동안 전면에서는 다른 작업을 동시에 할 수 있음] # tree 이용하기 (home에서 sudo snap install tree를 입력한 후) -하위 경로를 트리로 출력: tree # 파일의 사용권한 - 확인 방법: ls -sl (파일명.확장명) [예를 들어 ls -sl text.txt의 결과가 3 -rw-r--r--로 나온 경우] - 3은 파일의 블록 수 - 가장 왼쪽의 -는 일반 파일임을 의미함 [-: 일반파일, d: 디렉토리, l: 링크, p: 파이프, s: 소켓 등] - rw-의 -에는 x가 들어가는데 각각 r: 읽기, w: 쓰기, x: 실행을 나타냄 - rw-r--r--는 3개식 끊으면 rw- r-- r--, 이는 순서대로 소유자, 그룹, 기타의 사용권한을 나타냄 - 사용권한은 2진수와 8진수로 나타낼 수 있는데, rw-의 경우 110 = 6이 됨 #프로세스(process) - 실행 중인 프로그램 [주기억장치에 있어야 함] - linux에서 쉘(bash)는 항상 실행 중 - ps 명령어로 확인 가능 [프로세스 ID도 확인 가능] - kill 명령어로 강제 종료 가능 [kill (프로세스 ID)] # vi로 c 파일 만들기. 1. ubuntu에서 vi (파일명).c 입력 2. 왼쪽 아래에 "(파일명).c[new]" 가 표시되어 있을 때, 키보드의 i를 누르면 입력모드로 변환 (i외에도 a,o를 누르면 됨. i: 현재 위치에서 입력, a: 현재 위치 뒤에서 입력, o: 라인을 추가하고 입력) 3. 왼쪽 아래에 "--INSERT--" 가 표시되어 있는 것을 확인한 후, 쓰고싶은 c코드을 입력 4. 코드 작성후 키보드의 ESC를 누르고:wq!를 입력하면 저장하고 종료 (w: 작성 = write, q: 종료 = quit) 5. c파일이 생성된 것을 확인 가능
